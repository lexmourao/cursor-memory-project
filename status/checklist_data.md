# Data Checklist

> Starter checklist for projects created from the Cursor Memory Project template.  
> Use this only when the project includes source data, processed datasets, or structured data fields.

- [ ] All raw data stored in `data/raw/`
- [ ] Original/raw data remains immutable
- [ ] Processed outputs saved in `data/processed/`
- [ ] No personal data, credentials, secrets, PHI, or client-confidential data committed
- [ ] Data sources documented in the appropriate `docs/` file
- [ ] `docs/data_dictionary.md` updated if the project includes structured datasets or fields
- [ ] Data handling assumptions reviewed before using AI-assisted tools
