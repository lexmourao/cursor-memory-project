# Product Context

This project exists because long-running AI-assisted development work needs more than prompts.

It needs structured memory, clear scope, repeatable retrieval, summarization, test discipline, documented decisions, and human review.

Primary audiences:

- technical reviewers evaluating AI-assisted development workflows;
- developers studying local-first memory and retrieval patterns;
- AI specialists demonstrating supervised use of Cursor, ChatGPT, Claude, and related tools;
- future maintainers who need to understand what is active, archived, deferred, or intentionally out of scope.

The project’s value is the method: context quality, traceability, fallback behavior, local QA, and disciplined cleanup.
