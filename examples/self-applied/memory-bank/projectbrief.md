# Project Brief

Cursor Memory Project is a local-first AI-assisted development methodology repository.

It demonstrates how persistent project memory, retrieval workflows, summarization, lightweight backend APIs, documentation discipline, tests, and GitHub workflow hygiene can support long-running AI-assisted software work.

The project is maintained as a public technical portfolio artifact. It is not presented as a production SaaS platform, hosted RAG product, or enterprise compliance system.

Current purpose:

- demonstrate context engineering for AI-assisted development;
- preserve project memory across sessions;
- expose local memory and retrieval workflows through scripts and FastAPI slices;
- keep the repository auditable, testable, and honest about scope.
